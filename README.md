 Brand-Influencer Chat System (Real-Time Messaging)

## Overview
A real-time messaging module for brand and influencer communication.  
Supports chat, message delivery, and notification system.

## Features
- Real-time Chat (Firestore)
- Message Status (sent/read)
- Conversation History
- Push Notifications (Future scope)

## Tech Stack
- Frontend: React
- Backend: Firebase
- Database: Firestore
- Notifications: Firebase Cloud Messaging (Planned)

## Status
Currently in development. Initial structure created.
